# Udemy Clone Website

A responsive front-end clone of the Udemy website built using HTML and CSS. This project recreates the look and feel of Udemy's homepage, including the navigation bar, categories section, recommended courses, popular topics, and course cards.

## Features

* Responsive navigation bar with search functionality
* Course recommendation section
* Popular topics section
* Course cards with images, ratings, and pricing
* Clean and modern UI inspired by Udemy
* Responsive layout using Flexbox

## Technologies Used

* HTML5
* CSS3
* Flexbox

## Learning Outcomes

This project helped me strengthen my understanding of:

* Semantic HTML structure
* CSS styling and layouts
* Flexbox for responsive design
* Website cloning and UI replication
* Git and GitHub Pages deployment

## Project Status

Completed as a front-end practice project to improve web development skills and gain hands-on experience building real-world website layouts.
